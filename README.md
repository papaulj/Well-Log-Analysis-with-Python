# Well-Log-Analysis-with-Python

This project demonstrates how Python can be used to process and interpret well log data for reservoir characterization.

The project workflow is as below:
  Load the LAS well logs
  Clean the data/ perform brief data exploration
  Plot well logs
  Estimate porosity
  Identify reservoir intervals
  Detect net pay zones


Tools used for the project are:
●	Python
●	Pandas
●	NumPy
●	Matplotlib
●	Lasio
